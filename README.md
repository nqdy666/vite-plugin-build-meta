## Usage

Install

```bash
npm i vite-plugin-build-meta -D # yarn add vite-plugin-build-meta -D
```

Add it to `vite.config.js`

```ts
// vite.config.js
import VitePluginBuildMeta from 'vite-plugin-build-meta'

export default {
  plugins: [
    VitePluginBuildMeta()
  ],
}
```
